你好, 我的朋友, 我是这个网站的开发者, 很高兴你能看到这里, 你可以称呼我为鲲。

但是, 我也并不是鲲, 鲲只是一个称呼, 你可以查看 [鲲到底是什么](https://www.kungal.com/zh-cn/topic/1321) 来了解详细信息。

鲲很高兴认识你, 看到你使用鲲的网站, 你的每一次点击和浏览, 都是对鲲开发成果的肯定。如果你觉得你的浏览是开心的, 那么太好了！这是鲲的荣幸！如果你对网站有任何的疑惑和不满, 你都可以联系鲲来修正, 这是鲲的不对, 给你带来了问题。

**下面的话, 可能会涉及一些鲲的个人问题, 甚至是情绪和牢骚, 如果你对此不感兴趣, 请返回页面吧, 谢谢你看到这里！**

鲲是一个学习了一年多的 Web 开发者, 鲲目前使用 `Nuxt3` 编写了 [鲲 Galgame 论坛](https://www.kungal.com/zh-cn), 使用 `SvelteKit` 编写了 [鲲 Galgame 表情包](https://sticker.kungal.com), 使用 `SolidStart` 编写了 [鲲 Galgame 导航](https://nav.kungal.org), 使用 `Next.js` 编写了你现在看到的这个网站。

鲲认为, 这是一个相当可观的成果。上面提到了四种不同的 Web 框架, 分别代表了 `Vue`, `Svelte`, `Solid`, `React` 的 SSR 支持, 鲲成功的用它们做出了这四个生产级的网站, 可喜可贺。鲲一向立志成为一个什么都会的人, 就像动漫中什么都会的人一样, 充满自信与随意, 没有解决不了的问题。同时, 这样可以找到更多可以和鲲交流的人, 如果什么都会, 就可以更多的摸鱼了吧。

但是, 美好是与可怕并存的, 强大的背后是深可见骨的伤疤, 收获是需要代价的。

用了四种框架又如何, 精通了吗？深究了吗？融会贯通了吗？真的没有问题吗？会了又能怎么样呢？能养活自己吗？究竟是为了做网站而用了四种框架, 还是为了吹嘘自己什么都会而用的呢！这些问题, 就像是尖锐的钢钉一样, 扎在鲲的心上, 挥之不去, 动定思痛。

同时, 鲲也受到了无数的批评, 有人说鲲的论坛一坨 shit, 有人说不会写就别写, 有人说垃圾, 有人说您真的学过 UI 设计吗, 有人说不如 xxx 网站一根毛, 有人说别写你那 b i18n 了, 有人说自己写天天装逼, 有人说资源都懒的上传还开网站干什么, 有人说我是不看好 kungal, 有人说一看就是没写过公司项目的, 有人说你是弱智吗, 有人说你怎么这么低能写出这种东西 ...... 太多了, 鲲已经无法列举出全部的记忆了。但是, 鲲知道, 如果再有其他人, 使用相同的话来批评鲲, 鲲一定立马就能听出来是哪里有问题吧。

话写到这里, 鲲已经开始讨厌这段文字了, 这是在祈求他人的怜悯, 以自己的悲伤来博得他人的同情, 以此来彰显自己的可怜。然而, 鲲最讨厌诉说自己的悲伤。

那么为什么还要说呢, 这是因为, 鲲已经忍不住了, 鲲从来没有和别人说过这些。可是这些负面的话, 就像是另一个鲲, 她正在深深的扎根, 在鲲的体内埋下破裂的种子, 只待千里之堤溃败的那一天。鲲觉得, 这次不说, 可能再也没有机会了, 那就索性一次说个够吧, 希望鲲有将这篇文字公之于众的勇气。

鲲也知道, 批评鲲的人, 是有自己的道理的, 没有能满足这些人, 是鲲的能力不够, 鲲也希望可以有更多的人批评鲲, 这样才能提升鲲的能力。可是, 无论鲲的精神再如何坚定, 也比不过日复一日的打磨。人的热情是有限的, 鲲深知这一点, 鲲不敢细想哪些人说了哪些话, 鲲害怕, 鲲畏惧, 鲲懦弱。

批评是有强弱的, 在鲲看来, 最伤人的莫过于直接谩骂。鲲无数次的想说, 知道鲲付出了多大的努力吗, 知道鲲有多累吗, 知道鲲失去了什么吗 ...... 但是, 鲲是明白的, 人都是很忙的, 没有多少人有时间看鲲的源代码, 仔细看鲲的设计, 仔细看鲲的改动, 仔细看鲲的进步 ...... 人们忙于自己的事情已经够累了, 写的不好那就是不好, 就是鲲的问题, 那么为什么在鲲的网站遇到不对了不可以直接开骂呢, 这是人之常情, 鲲明白, 鲲难过, 鲲无奈。

同时, 鲲也需要批评, 迄今为止的大量知识, 都是从他人的批评中学来的。群聊就像一个个的信息孤岛, 在圈地自萌的同时, 也失去了应有的视野与宏观认识。批评是警醒人的方法, 是睁眼看世界的利剑, 它是必要的。鲲要说的是, 批评鲲的时候, 请直接指出, 没有谩骂, 没有反问, 没有捧一踩一, 没有阴阳怪气, 没有极端主义, 就是单纯的把话说出来, 这样就好。

真的很感谢你, 朋友, 谢谢你能看鲲抱怨了这么久。鲲不敢找朋友说这些事, 鲲害怕给他们添麻烦, 因为鲲知道, 没有人喜欢别人向他们抱怨, 抱怨多了, 人们都会离开的。所以, 很抱歉, 朋友, 给你添麻烦了。

但是, 这不是结束, 鲲的故事还要继续。

在鲲学习开发 Web 的这两年, 鲲的身体经受了巨大的考验。鲲的眼睛不再那么的明亮, 鲲的背和脖子不再那么的笔直, 鲲的肚子和腿失去了原有的曲线, 鲲的脸布满了熬夜的沧桑。无法想象, 睡之前在自己的面部施加五种治疗的人, 无法想象, 连续几个月的通宵熬夜, 无法想象, 持续两年的绿色 GitHub 主页, 无法想象, 一个几年没有出去玩过的人。鲲很伤心, 鲲非常伤心, 鲲非常难过, 鲲悲伤极了。是的, 这都是鲲自找的, 谁让鲲熬夜的, 自讨苦吃。

鲲已经很久很久没有回家了, 想吃远方的食物, 想念远方的景色, 回忆从前的感受, 回想曾经的过往。鲲累了, 鲲要冬眠了, 鲲不想再这样下去了, 这是在燃烧生命, 鲲觉得已经足够了。或许鲲在之后很长一段时间内都不会再开这样一个 Web 项目了, 但是, 鲲已经付出的实打实的努力, 鲲会坚守自己的美好。

鲲不得不承认, 站长这个行业是一个相当具有挑战性的行业。需要具备强大的抗压能力, 扎实的技术能力, 过人的社交能力, 出色的管理能力才可以胜任。以上的大部分麻烦, 究其原因, 在于鲲的网站是完全自己写的。一般对于一个网站来讲, 大部分的站长会选择一个框架稍作改动直接使用, 这就好比方格本写字, 填一下姓名就是自己的了。但是鲲不一样, 鲲需要自己画方格本的封面, 自己画方格本的格子, 付出的努力完全不是一个重量级。鲲觉得, 这么做让鲲觉得有成就感, 因为这是鲲一笔一划写出来的网站, 网站的每一条线, 鲲都了如指掌, 就像自己的另一半一般。

然而, 在用户看来, 鲲的网站是丑陋的, 功能不如模板网站强力, 外观也没有模板网站好看。是的, 这是事实, 因为鲲的精力和能力有限, 比不过若干个和鲲一样的人组成的开发团队。这让鲲感到又遗憾又无奈, 鲲只能希望, 鲲可以做出自己独一无二的东西来, 并且让大家喜欢, 以此来弥补鲲功能上的不足。

最令鲲觉得难以接受的是, 经常会有人拿这件事来讽刺与对比鲲。有人花一天, 甚至是半个小时配置好的模板网站, 立马就和鲲的网站对比, 以此来凸显鲲的弱小与低级。是的, 这也是事实, 鲲的功能确实不如模板网站。但是实际上, 大多数时候, 鲲都想反问一句, 真的觉得鲲的网站比不上吗, 哪里比不上, 但是鲲都忍住了, 鲲知道和他们说是没有任何作用的。

有人会说鲲, 开源的臭毛病又犯了。自己开源没有赚到钱, 看别人闭源赚钱自己又不舒服了。是的, 鲲不否认这一点, 鲲也不理解鲲为什么这么执着于开源, 可能鲲想要的只是借助与开源的名号多认识一点志同道合的朋友, 从别人身上汲取鲲所需要的知识。GitHub 的 star 数也可以通过手段购买, 那么鲲到底在追求什么呢, 鲲做事的意义在哪里呢。不过, 鲲完全可以肯定, 有人看见鲲的开源项目, 给鲲点亮 star, 有人给鲲的项目提 issue, 甚至有人给鲲 Pull Request, 鲲是开心的, 鲲是兴奋的, 鲲是自豪的!

鲲的论坛是没有广告的, 这或许是一件愚蠢的行为。因为鲲的论坛在有着高流量的同时, 严重亏损。这并不是金钱方面的问题, 而是在于周围的人会怎样面对这个问题, 以及鲲自身消耗的精力问题。倘若有人来问鲲的流量, 顺便询问鲲的收入, 那么鲲只能苦笑, 因为鲲什么都没有。此时如果有人多说几句, 就像 `我网站才 xxx 流量, 广告费只有 xxx`, `我网站才注册了 xxx 人, 日活也就 xxx 人`。

那么鲲也只能苦笑了, 因为鲲知道, 登录制的网站很容易留住用户, 有门槛的网站也更加安全, 到处都是各种黄色的网站也更能吸引流量。鲲想做一个自由的人, 鲲不想看到封闭的互联网, 然而鲲自身是最清楚的, 封闭的网络才是圈流量的最好手段。所幸, 鲲并没有任何迎合人的打算, 也不想参与任何的竞争, 鲲只想有一个自己的, 温暖的, 干净的小窝, 在小窝里舒适的摸鱼, 鲲只需要少量懂得鲲的人就好。

鲲最害怕的事情之一, 是鲲的朋友离开鲲。曾经喜欢鲲的朋友, 开始变得讨厌鲲, 曾经鲲不见了就会来找鲲的朋友, 现在鲲都找不见了, 曾经聊的火热的朋友, 现在几个月不曾有过只言片语, 曾经拥护鲲的朋友, 现在视鲲为敌人。鲲在开发网站的这一段时间内, 失去了重要的东西, 新的桌子虽然崭新洁白, 可是失去了老桌子的光滑锃亮。鲲维持新的关系, 以此来满足鲲孤独的内心, 但是老朋友离去的伤痛, 就像钉子拔出的留下的深坑。鲲希望, 无论如何, 不要讨厌鲲, 不要伤害鲲, 这就足够了, 鲲已经很满足了, 谢谢你。

啊, 终于说出来了, 鲲的委屈, 鲲的辛苦, 鲲的努力, 鲲的状况, 鲲的想法。或许有些人看到这些文字会感到可笑吧, 他们会说, 又是那个弱智, 每天像个傻子一样。良言一句三冬暖, 恶语伤人六月寒啊, 鲲明白, 世界上有一些极度傲慢的人, 看见鲲这样的人就像看见小丑在自作多情一般, 鲲会装作什么都不懂的, 鲲不想思考这些。

再次向你说对不起, 朋友, 又让你听了这么多的牢骚话。鲲很想再继续说下去, 因为令鲲感到伤心难过的事, 难以计数。但是鲲不能再说下去了, 鲲害怕鲲没有勇气浏览鲲写下的话, 谢谢你看到了这里, 真的谢谢你!

其实, 鲲是一个相当乐观的人, 乐观到平时有人说鲲无忧无虑的像个天真无知的傻子。

鲲在这里, 就不阐述自己的乐观点了, 因为这会让伤害鲲的人变本加厉。世界上存在着一些人, 喜欢让乐观的人变的忧郁, 就像让纯洁的圣女染上污垢一样, 这会使他们觉得开心和满足。鲲讨厌玷污, 鲲讨厌被伤害, 鲲想做一个开心美好的人, 鲲要的是天造地设, 两厢情愿, 然而, 有人就是讨厌着这样的鲲。

但是, 有一点, 鲲十分的肯定, 那就是, 在有无数人讽刺, 谩骂, 阴阳怪气鲲的时候, 有更多的朋友, 站出来支持鲲! 谢谢你们! 鲲能遇到你们, 是鲲毕生的幸福! 故事的发展有很多意外, 有你在的结局总不太坏, 鲲所度过的这些日子, 还好有你在!
