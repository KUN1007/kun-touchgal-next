## 为什么要发布 Galgame

鲲 Galgame 补丁 采用与 [鲲 Galgame 论坛](https://www.kungal.com) 同样的 [cfmsc 发布机制](/about/notice/cfmsc), 以此来确保资源的唯一性和用户的合作贡献

因此, 要发布一个 Galgame 补丁下载资源

> 若不存在这个补丁资源对应的 Galgame, 您需要先前往 [创建新游戏](/edit/create) 页面创建一个对应的 Galgame, 然后再将补丁添加到新建的 Galgame 下

> 若已经存在补丁资源对应的 Galgame, 直接将您想要发的补丁资源添加到这个 Galgame 的底下即可

## 发布 Galgame 流程

发布 Galgame 需要下面的几个流程

查找游戏的 VNDB_ID (例如 v19658) -> 检查游戏是否重复

-> 若重复, 请直接搜索并前往这个游戏的资源页面添加补丁下载资源

-> 若不重复 -> 添加游戏的预览图 -> 按您的喜好编写游戏介绍以及标签 -> 点击发布

### 查找游戏的 VNDB_ID

关于这一步骤可以查看这篇文章

[如何通过 VNDB 检索 Galgame?](https://www.kungal.com/zh-cn/topic/1040)

填写 VNDB_ID 后, 点击 **检查重复** 按钮, 即可自动检查该游戏是否重复

在检查重复之后, 系统会为您自动获取游戏的别名数据

### 添加游戏的预览图

对于 PC 端, 点击选择文件或者拖拽您想添加的图片即可

选择图片之后, 会弹出一个裁剪图片的提示, 请裁剪您的图片到 `16 : 9` 的比例

裁剪完成后的图片会被强制调整大小到 `1920 × 1080` 分辨率, 这是为了美观考虑

点击确定, 即可完成裁剪

### 编写游戏介绍

您可以使用 Markdown 格式编写游戏的介绍, 也可以插入图片

需要注意的是, 每个用户每天可以上传 50 张图片, 不要插入 R18 图片

关于 Markdown 的写法可以查看 [论坛编辑器语法指南](https://www.kungal.com/zh-cn/topic/1225)

编辑器可能产生令人困惑的行为, 如果您遇到任何问题, 欢迎 [联系我们](/about/notice/feedback)

### 添加游戏别名

游戏的别名是自动添加的, 但是为了更良好的性能, 我们十分推荐 **将第一个别名写为游戏的日文原名**

如果您觉得这些别名不能满足您的需求, 您可以编写别名, 按下回车, 或点击 `+` 添加游戏别名

### 点击发布

当您完成了上面的步骤, 直接点击发布即可

如果您填写的信息出现了问题, 网站顶部会有报错提示

如果发生您无法解决的问题, 欢迎 [联系我们](/about/notice/feedback)
