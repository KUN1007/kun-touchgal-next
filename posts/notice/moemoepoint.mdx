## 萌萌点是什么

萌萌点是本站唯一是通用虚拟货币, 仅供网站社区使用, 您永远不可能为萌萌点花费任何现实资产, 现实资产也永远不可购买萌萌点

我们认为, 萌萌点的多少取决于他人对您的赞赏, 因此, 给予无偿奉献的朋友一个点赞, 是给予他们最真诚的感谢

萌萌即美好

## 有哪些需要注意

初始注册您的萌萌点为 0

## 如何获取萌萌点

1. 已注册用户每日签到会随机获得 0 ~ 7 萌萌点

2. 评论, 补丁资源 被其他人点赞, 您将获得 1 萌萌点

3. 其他人收藏了您发布的 Galgame, 您将获得 1 萌萌点

4. 发布一个 Galgame, 您将获得 3 萌萌点

4. 发布一个 Galgame 补丁资源, 您将获得 3 萌萌点

## 如何消耗萌萌点

1. 更改用户名需要 30 萌萌点 （因为实在想不出来萌萌点有什么用了）

2. 其他人取消点赞 / 收藏您的 Galgame / 补丁资源 / 评论, 您得到的 1 萌萌点会同步被扣除

3. 删除评论 / 资源, 评论 / 资源被点赞所得到的萌萌点将会被扣除

4. 删除一个 Galgame 补丁资源, 您发布获得的 3 萌萌点也会被扣除

## 之后有什么想法

因为我们也不知道萌萌点可以做什么, 如果您有好的想法, 欢迎 [联系我们](/about/notice/feedback)
